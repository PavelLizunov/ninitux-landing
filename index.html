how it worksкак это работает

  1. Install — run the one-liner above for your OS, or grab a manual build from Releases.

    Windows needs admin. macOS needs a one-time sudoers entry. Linux uses POSIX capabilities.

    Установить — выполните команду для вашей ОС или скачайте сборку с Releases.

    Windows требует admin. macOS — однократный sudoers. Linux — POSIX capabilities.

  2. Open the app — paste a subscription URL, drop in your own sing-box JSON, or use the Free tab.

    Subscriptions and the public pool merge into one server list. One-click TCP+TLS test on any server.

    Откройте приложение — вставьте subscription URL, свой sing-box JSON или используйте вкладку Free.

    Подписки и публичный пул сливаются в один список серверов. Тест TCP+TLS в один клик.

  3. Pick which apps go through the tunnel — selected processes route through the proxy, everything else stays direct.

    ETW watches new processes and hot-reloads the sing-box config without reconnecting.

    Выберите приложения для туннеля — выбранные процессы идут через прокси, всё остальное напрямую.

    ETW следит за новыми процессами и горячо обновляет конфиг sing-box без переподключения.
